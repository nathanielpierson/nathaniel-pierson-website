# Nathaniel Pierson Portfolio

React + Vite portfolio website for showcasing projects with deep project pages, screenshots, and demo video embeds.

## Local setup

```bash
npm install
npm run dev
```

## Projects content

Edit `src/data/projects.js` (slugs, screenshots, captions, GitHub links, and optional `youtubeUrl`).

## View counter

View counter feature has been removed from this project.

## Deploy

```bash
npm run build
```
